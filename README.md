# Siyana Card

A personal business-card landing page — the first live piece of the **Siyana Code** brand.

## Live Demo
https://siyana-code.github.io/siyana-card/

## Tech Stack
- HTML5
- CSS3 (Flexbox)

## What I Practiced
- Semantic HTML structure (`main`, `nav`, `a`)
- CSS class selectors and scoping
- Flexbox centering
- Deploying a static site with GitHub Pages

## Run Locally

2. Open `index.html` with Live Server (or any static server)
